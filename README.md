# Predictive Maintenance AI

Motor fault detection using Machine Learning.

## What it does
- Detects motor faults using temperature, vibration & current data
- 88% accuracy using Random Forest
- Built with Python, Scikit-learn, Pandas

## Tools Used
- Python
- Scikit-learn
- Pandas, NumPy
